# Fluxo de Trabalho STAKIM

## Fluxo padrão

CEO → Secretária → Coordenador → Diretores → Diretor de Qualidade → Secretária → CEO

## Regras

- A Secretária participa de reuniões importantes.
- O Coordenador distribui tarefas.
- Diretores executam dentro de suas áreas.
- Diretor de Qualidade revisa entregas críticas.
- Diretor de Conhecimento registra decisões.
- Diretor de Tecnologia controla Git, terminal, VS Code, Claude Code e infraestrutura.
