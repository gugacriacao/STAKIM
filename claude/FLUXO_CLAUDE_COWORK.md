# Fluxo Claude Cowork

## Papel do Claude Cowork

O Claude Cowork é o escritório operacional da STAKIM.

## Como usar

1. Abrir a pasta da STAKIM.
2. Pedir para ler `INSTRUCOES_PARA_CLAUDE.md`.
3. O Claude executa etapas.
4. O CEO acompanha no Claude e no Painel.
5. Decisões são registradas.

## Regra

O Painel organiza. O Claude constrói. O GitHub guarda.
