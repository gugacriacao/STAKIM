# Operação STAKIM

Rotinas, processos operacionais e checklists do dia a dia — gerenciado pelo Diretor de Operações (ver `.claude/agents/diretor-operacoes.md`).

## O que entra aqui

- Checklists de rotina.
- Runbooks operacionais (como fazer X passo a passo, fora do escopo de `processos/`).
- Registros de eficiência/organização interna.

## Diferença para `processos/`

`processos/` define as regras gerais (fluxo de trabalho, reuniões, aprendizado contínuo). `operacao/` guarda a execução prática dessas regras no dia a dia.

Pasta ainda vazia — primeiro conteúdo entra quando houver rotina operacional formalizada.
