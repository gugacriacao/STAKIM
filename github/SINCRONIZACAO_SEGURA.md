# Sincronização Segura com GitHub

## Objetivo

Permitir que o CEO trabalhe de qualquer lugar com segurança.

## Botões futuros do Painel

### Enviar alterações
Executa fluxo seguro de commit e push.

### Atualizar projeto
Executa fluxo seguro de pull.

## Antes de enviar

- Verificar arquivos modificados.
- Criar resumo.
- Rodar testes quando existirem.
- Criar commit.
- Puxar alterações remotas com segurança.
- Enviar.

## Antes de atualizar

- Verificar alterações locais.
- Se houver conflito, parar e pedir revisão.

## Regra

Nunca sobrescrever trabalho local sem aprovação.
